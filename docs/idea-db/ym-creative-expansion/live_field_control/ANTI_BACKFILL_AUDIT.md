# Anti-backfill audit

A patch fails if it uses any of the following to discharge an upstream field:

- final CMP116 Lemma 3 bound as proof of Wilson Hessian identification;
- Appendix-F H# identity as proof of Gaussian pushforward;
- scalar Dimock covariance/root architecture as a Yang-Mills covariance-root theorem;
- decay estimate as proof of support containment or measurability;
- Rsc/flow bound as proof of local activity construction.

The patch may cite downstream material only as a consumer or consistency check,
not as source evidence for the upstream field.
