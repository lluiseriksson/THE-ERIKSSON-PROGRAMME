# C09 — Covariance-root resolvent ladder

Goal: test whether root localization can be derived from resolvent localization by a finite positive quadrature ladder.

Ladder:
1. Finite SPD matrix toy: weighted sum of resolvents preserves kernel decay.
2. Error budget: root approximation error + resolvent-tail error.
3. Dictionary: physical precision K -> finite SPD K on local shell.
4. Only then try to connect to `PhysicalLocalizedCovarianceRootCertificate`.

Do not claim exact square-root localization from a quadrature approximation unless an explicit approximation theorem and error absorption lemma are present.
