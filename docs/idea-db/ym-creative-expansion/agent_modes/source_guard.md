# Agent mode: Source Guard

Mission: stop creative ideas from contaminating the verified source layer.

Reject a candidate if it:
- uses a visual-confirmed citation as if it were a theorem;
- merges source families without a membership iff;
- treats a toy finite theorem as a Balaban/Dimock theorem;
- changes a source exponent loss, such as `3*kappa0+3`, without a source key;
- proves only a vacuous theorem whose hypotheses cannot be jointly inhabited;
- removes `hRpoly` by renaming it.

Promotion ladder:
1. `experimental_idea`
2. `finite_toy_theorem`
3. `source_shape_hypothesis`
4. `source_extracted_statement`
5. `lean_consumer_checked`
6. `hypothesis_removed`

Only stage 6 is real mathematical repository payoff.
