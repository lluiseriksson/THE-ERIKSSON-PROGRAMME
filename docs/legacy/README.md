# Legacy archive

Everything in this directory predates the **2026-05-29 cleanup** (see
`/CLEANUP_PLAN.md` and `/FOUNDATIONS.md`) and is kept **only as a
historical record**. None of it describes the current state of the
project, and none of it is load-bearing: the verified development is
`YangMillsCore` and its documentation lives at the repository root and
in `/docs`.

In particular:

* `AXIOM_FRONTIER.md`, `SORRY_FRONTIER.md`, `STATE_OF_THE_PROJECT.md`,
  `UNCONDITIONALITY_ROADMAP.md`, `PHASE8_*.md`, `ROADMAP_MASTER.md`,
  `AI_ONBOARDING.md`, `DECISIONS.md` — status documents from the era
  when the repository's terminal theorem was the **vacuous**
  `∃ m > 0` target. Their percentages, axiom censuses and plans
  measure progress toward that target and are therefore meaningless
  for the current programme.
* `registry/`, `dashboard/`, `papers/` — a YAML dependency-registry
  workflow and paper index from the same era. The current project
  records state in `docs/VERIFICATION-LEDGER.md` (machine-checked
  oracle outputs) instead.
* `*.py`, `*.ps1` — deployment and audit scripts for workflows that
  no longer exist.

For the current state of the project, start at the root `README.md`.
